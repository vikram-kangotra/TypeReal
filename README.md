# TypeReal
A text to hand-writing python utility to complete your assignment
# Usage
'''
usage: typeReal.py [-h] -i IFILE -o OFILE

Text to Handwriting converter

optional arguments:
  -h, --help            show this help message and exit
  -i IFILE, --ifile IFILE
                        The input file
  -o OFILE, --ofile OFILE
                        The output file
'''
